# Passo 01 — Início e planeamento

Data: 2026-04-08

## Objetivo
Iniciar o projeto e definir os próximos passos para criar um portal imobiliário em PyQt6 ligado à base de dados já existente.

## O que foi feito
1. Criação da estrutura inicial do projeto:
   - `src/`
   - `docs/passos/`
2. Criação de documentação base no `README.md`.
3. Definição do processo incremental (passos pequenos + validação).

## Próximo passo (Passo 02)
Mapear a base de dados e preparar a camada de acesso a dados.

## Informação necessária do utilizador
Para avançar, é necessário:
- Tipo de base de dados (SQLite, MySQL, PostgreSQL, SQL Server, etc.)
- Método de ligação (ficheiro local, host/porta, credenciais)
- Estrutura principal (tabelas e campos)
- 3 a 5 operações prioritárias da primeira versão (ex.: listar imóveis, pesquisar por localização, criar anúncio)

## Critério de conclusão deste passo
Projeto inicial criado e documentação do passo registada.
